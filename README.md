# Store_Products_Manager

A software that can be used to manage the inventory of any store.
The software includes the following features - "Product addition", "Product deletion", "Product sorting","Product search engine", "Action cancelation" - same as CTRL+z,"Message sending" to existing customers.
The program required the usage of threads and included different design patterns such as - Observer, Singleton, Memento, and more.
The program was written in JAVA and JavaFX.
